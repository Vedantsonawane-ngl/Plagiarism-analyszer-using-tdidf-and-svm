🔹 Plagiarism Checker (TF-IDF + SVM)

Developed a machine learning-based plagiarism detection system that analyzes textual similarity between documents. The project uses TF-IDF vectorization to convert text into numerical features and applies a Support Vector Machine (SVM) model to classify whether content is plagiarized or original. It efficiently compares multiple documents and highlights similarity scores, helping identify copied content. This project strengthened my understanding of NLP, feature extraction, and classification algorithms.
